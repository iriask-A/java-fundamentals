# java-fundamentals

A repository for learning and practicing core Java concepts, including lab assignments, practice tasks, and exercises.

## 📁 Repository Structure

```
java-fundamentals/
├── lab1/           # Laboratory work 1
├── lab2/           # Lab 2 assignments
├── practice/       # Basic Java tasks
├── practice1/      # Basic tasks pt2
├── practice2/      # Assignment 2 practice
├── practice3/      # Practice 3 exercises
├── First.java      # Multiplication example
└── README.md
```

## 🚀 Getting Started

### Prerequisites

- [Java JDK 11+](https://www.oracle.com/java/technologies/downloads/)
- [IntelliJ IDEA](https://www.jetbrains.com/idea/) (recommended) or any Java IDE

### Running a File

```bash
# Compile
javac First.java

# Run
java First
```

## 📚 Topics Covered

- **Lab Work** – Structured laboratory assignments (`lab1`, `lab2`)
- **Practice** – Hands-on Java exercises covering basic to intermediate concepts
- **Core Java** – Variables, loops, conditionals, methods, OOP principles
